# Two-Spacecraft Formation Flying and Impulsive Burn Analysis Using GMAT
Configured and analyzed a two-spacecraft mission scenario using the General Mission Analysis Tool (GMAT) by defining spacecraft parameters, force models, numerical propagation, and impulsive maneuvers with trajectory visualization using Orbit View and XY plots.
## Overview
This project was developed using the General Mission Analysis Tool (GMAT), an open-source space mission analysis software originally developed by NASA.
It demonstrates an introductory two-spacecraft mission involving orbital propagation, formation flying, and impulsive burn simulation. The mission includes spacecraft configuration, force models, numerical propagation, and visualization using Orbit Viewand XY Plot.
## Features
- Two-spacecraft LEO mission
- Formation setup
- Impulsive burn
- Numerical propagation
- Orbit View visualization
- Ground Track Plot
- XY Plot
## Software
- NASA GMAT R2026
